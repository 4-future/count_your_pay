# 你的花费

使用 Flutter 开发的客户端，包括：Mac、Web、iOS、Android 平台。

## 介绍

一个简单的记账软件，试点项目。

## 目的

学习 Flutter 的 Demo 项目。


## 安装

依照 Flutter 官网，进行 flutter 安装，配置后执行 `flutter doctor -v`，查看环境配置结果。

## Flutter 相关使用介绍

### 核心

- Tab
- Stack
- 路由跳转 + 参数传递
- 有状态、无状态
- 常用组件
- 样式
- 动画
- 全局参数

### Basic widgets

- Text：创建一个带有样式的文本
- Row、Column：创建 flexible 布局的 widgets
- Stack：基于Web绝对定位布局模型
- Container：创建一个长方形可视区域
